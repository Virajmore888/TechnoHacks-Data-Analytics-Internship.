## 📌 Executive Summary
The primary objective of this project was to implement a robust Data Cleaning and Exploratory Analysis (EDA) framework on a 2025 sales dataset. By utilizing advanced Excel methodologies, I successfully sanitized raw data and established key performance indicators (KPIs) to evaluate organizational growth and transaction efficiency.

## 📖 Technical Data Dictionary
| Attribute | Functional Description | Data Type |
| :--- | :--- | :--- |
| **Order Date** | Temporal marker for transaction logging. | Date |
| **Category** | High-level product segmentation. | String |
| **Units** | Volumetric count of items per transaction. | Integer |
| **Total Revenue** | Final financial output (Units × Unit Price). | Currency (₹) |
